Game Information

Rules

Jungle Relics is a cascading slot game where winning symbols disappear and new symbols fall into place, creating opportunities for multiple wins in a single spin.

How to Play

  • Set your bet amount using the bet adjustment button
  • Click SPIN to start the game
  • Match 3 or more symbols horizontally to win
  • Winning symbols cascade and new symbols fall into place
  • Free spins can be triggered by scatter symbols or purchased

Paytable

Symbol values are calculated based on your bet amount. Higher value symbols pay more when matched.

Scatter Symbol: Triggers free spins when 3 or more appear

Wild Symbol: Substitutes for other symbols to create winning combinations
